# 2D Game with Phase.js

A 2D platform game made with Phase.js

## Setup
1. Install the http-server.

```bash
npm install -g http-server
```

2. Run the server.

```bash
http-server
```

3. In your browser open the page:
```
http://localhost:8080/
```
